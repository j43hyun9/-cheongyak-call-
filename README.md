# 청약콜 (cheongyak-call)

> AI 음성비서 + 청약알림 + 일정관리

마이크에 대고 말하면, 공모주(청약) 일정과 개인 일정을 **음성과 텍스트로 안내**하는 AI 음성 비서 데모입니다.

## 주요 기능

- 🗣️ **음성 명령** — 버튼을 누르고 말하는 PTT(Push-to-Talk) 방식
- 📈 **청약 일정 안내** — 오늘 / 이번 주에 청약 가능한 공모주 안내
- 📅 **일정 등록** — "내일 오후 3시에 팀 회의 등록" 처럼 음성으로 일정 저장
- 🔎 **일정 조회** — "내일 일정 알려줘" 로 등록한 일정 확인
- 🔊 **음성 + 텍스트 응답** — 모든 응답을 화면과 음성으로 동시 제공

## 기술 스택

| 구분 | 사용 |
|------|------|
| 언어 | Python 3.10+ |
| UI | Gradio |
| 음성 인식 (STT) | OpenAI Whisper |
| 음성 합성 (TTS) | gTTS |
| 청약 정보 수집 | requests + BeautifulSoup |

## 실행 방법

```bash
pip install -r requirements.txt
python app.py
```

실행 후 브라우저에서 `http://127.0.0.1:7860` 접속.

> Whisper 사용을 위해 시스템에 **FFmpeg** 설치가 필요합니다.

## 사용 예시

| 말하기 | 응답 |
|--------|------|
| "오늘 공모주 알려줘" | 오늘 청약 가능한 공모주를 안내 |
| "이번 주 공모주" | 이번 주 청약 일정 안내 |
| "내일 오후 3시에 팀 회의 등록" | 일정 저장 후 확인 안내 |
| "내일 일정 알려줘" | 내일 등록된 일정 안내 |
